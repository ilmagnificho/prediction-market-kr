# 예측마켓 (Korean Prediction Market)

한국형 예측 마켓 플랫폼 MVP

## 🎯 프로젝트 소개

Polymarket, Kalshi와 같은 글로벌 예측 마켓을 한국 사용자에게 맞게 재해석한 플랫폼입니다.
단순한 betting이 아닌, B2B consumer market research data platform으로의 성장을 목표로 합니다.

## ✨ 주요 기능

- **간단한 YES/NO 투표**: 누구나 쉽게 참여 가능한 예측 투표
- **실시간 결과 집계**: 투표 현황을 실시간으로 확인
- **커뮤니티 기능**: 각 투표마다 댓글과 토론 가능 (한국 사용자들이 좋아하는 '댓글 놀이')
- **카테고리별 분류**: 경제, 스포츠, 테크, 부동산 등
- **모바일 최적화**: 심플하고 직관적인 UI/UX

## 🚀 시작하기

### 로컬에서 실행

1. 저장소 클론
```bash
git clone https://github.com/YOUR_USERNAME/prediction-market-kr.git
cd prediction-market-kr
```

2. 브라우저로 열기
```bash
# 단순히 index.html 파일을 브라우저로 열면 됩니다
open index.html  # MacOS
start index.html # Windows
```

### Vercel로 배포

1. [Vercel](https://vercel.com)에 로그인
2. GitHub 저장소 연결
3. 자동으로 배포 완료!

## 🛠 기술 스택

- **Frontend**: React (via CDN)
- **Styling**: Tailwind CSS
- **Storage**: LocalStorage (MVP)
- **No Backend Required**: 정적 사이트로 동작

## 📱 사용 방법

1. **홈 화면**: 진행중인 투표 목록 확인
2. **투표 참여**: 원하는 투표 클릭 → YES/NO 선택
3. **댓글 작성**: 투표 상세에서 의견 공유
4. **새 투표 만들기**: 우측 하단 + 버튼
